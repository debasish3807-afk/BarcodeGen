# Changelog

## [1.0.0] - 2025-07-06

### Added
- **Barcode Generator** - 11 formats (Code 128, Code 39, Code 93, EAN-8, EAN-13, UPC-A, UPC-E, ITF, Codabar, MSI, GS1-128) with live preview, full customization, and 10 export formats
- **QR Code Generator** - 10 types (URL, Text, Email, Phone, SMS, WhatsApp, WiFi, vCard, Location, UPI) with module/eye styles, gradients, logo upload
- **Batch Generator** - CSV/Excel/TXT/JSON/Manual input, progress tracking, pause/resume, ZIP downloads (10,000+ items)
- **Templates** - 18 professional presets for barcodes and QR codes
- **QR & Barcode Scanner** - Camera and image upload scanning with history
- **Favorites** - Local storage with add/remove/rename/duplicate/import/export
- **History** - Recently generated items with search/filter/delete/export
- **AI Generator** - Pattern-matching engine, 17 patterns, instant field preparation
- **Blog** - 10 articles, 9 categories, search, related posts, share buttons
- **Multi-Language** - 20 locales with i18n architecture and RTL support
- **Developer Console** - API keys, usage analytics, playground, 9 SDKs, logs, webhooks
- **Admin Dashboard** - Users, analytics, blog, templates, settings management
- **Enterprise Backend** - Prisma (18 tables), REST API (10 endpoints), JWT, RBAC, Redis
- **DevOps** - Docker, Docker Compose, Nginx, CI/CD, Prometheus, Grafana
- **Email** - SMTP/SendGrid/Resend with 5 HTML templates
- **Storage** - AWS S3/Cloudinary/Backblaze/Local abstraction
- **Security** - OWASP headers, CSP, rate limiting, encrypted keys, webhook verification
- **PWA** - Service worker, manifest, offline support
- **SEO** - Schema.org, breadcrumbs, sitemap, robots.txt, OG/Twitter cards
- **System Status Page** - Real-time service health monitoring
- **Toast Notifications** - Success/error/warning/info with auto-dismiss
- **Error Handling** - Custom 404, 500, global error boundary, loading skeletons
